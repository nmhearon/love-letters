I tried to see you through my windows, but had to use a mac until I figured out how I could get to you. While at first you made me nervous, I've grown to realize how easy you can make things. When I open command prompt, I call your name "bash". Sometimes I get confused about where we are in our relationship, but pwd always lets me know exactly where we are. Together, we can see the world around us ls. All I have to do is pop in a cd, and you're my copilot getting me to where I wanna go. But with mkdir we can build our own place. And then when we finally made our own place, I can touch you.html and when I do we've created a whole new file together.

And while you can do so many useful things like that, you can also make me laugh with all your little tricks. With telnet blinkenlights.nl you're playing Star Wars; with no words unfortunately, but Star Wars nonetheless. And if I ever need help, I know it's only four letters away.

Common Commands:

  • touch index.html: Will create a file in the current directory named index.html
  • pwd: Will print the name of the current working directory
  • cd: Will change the working directory to whatever follows it
  • ls: Lists the files in the current working directory
  • ls -la: Lists ALL files within the current working directory including hidden files
  • mkdir: Makes a new directory within the current directory
  • help: Will populate a list of common commands
  • clear: Will clear the current workspace so it appears to be a fresh window and lacks clutter
