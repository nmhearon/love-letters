Dear Git,

While I've never been horrible at keeping track of all the changes in my life (and code), you certainly make things easier. All it takes is a little git init to turn you right on. And once I do, a simple "git add ." is enough for you to recognize all the changes I'm making. When you want to make notes entailing our story together, you can always git commit them right to memory, and for special events, -m "will help you to write a special note to yourself recounting the details". But of course, couples need to check in with each other every now and then, and git status helps me to know exactly how you're doing and if you're keeping up to date with me. Whenever I want to remember all the good times we've shared, git log shows me everything we've been through together. And while we both may have different branches of our lives, we can always make a new one together with git branch name a new one for ourselves. And once we decide things are going well enough, with a simple git merge branches we can take our separate branches and make them one to share everything that we have together. git push will allow us to share our union and contributions to each other with the world thanks to our dear friend GitHub. And if there comes a sad day where we no longer see ourselves together, git branch -D will delete our history completely, and git checkout allows us to revert to our lives before we made all our changes together. But I hope our bond is one that will never break.
