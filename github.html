Common Commands:

  • git add .: This command is used to add all of the files from your local directory to your git repository
  • git commit: This commits your changes to your code to the repository you've created after you add your files to it; note that following this command with "-m "notes here"" will add the notes you've made to your update to help track your updates
  • git status: This command will tell you the current status of your repository and alert you as to whether or not it is up to date
  • git log: This command will show you all of the updates you have commited to your repository
  • git remote add origin githuburl: This command will add your remote GitHub repository by the githuburl you enter so that you can push updates to it
  • git push: This command will allow you to push your repository to your remote GitHub repository
